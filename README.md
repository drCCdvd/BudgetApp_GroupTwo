# ButJet

ButJet is a simple and modern budgeting app designed to help users gain clarity over their personal finances and build healthy money habits over time. Primarily it is possible in the app to create goals.
When goals are created you can then see how close you are on reaching them.

The focus is on simplicity, usability, and reducing financial stress.

## Features
- Simple budget overview
- Track income and expenses
- Offline-first
- Clean and intuitive user interface
- All data stored locally on the device

## Purpose
AppName is built for users who want to:
- Manage a personal budget without complexity
- Keep their financial data private
- Use a budgeting app without accounts, servers, or cloud sync

## Technology Stack

### Platform
- Android
- Kotlin

### UI
- Jetpack Compose
- Material Design

### Architecture
- MVVM
- Unidirectional data flow

### Data Storage
- Local file storage
- JSON files stored on the device
- No database, No backend, No external services

### Tooling
- Gradle
- Android Studio
